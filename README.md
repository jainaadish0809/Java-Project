# 📁 File Management System (Java)

## 📌 Project Overview
This project is a console-based File Management System developed in Java.  
It allows users to perform various file and directory operations such as creating, deleting, reading, writing, and searching files.

The system is designed using Object-Oriented Programming (OOP) principles and implements File I/O handling, exception handling, and modular architecture to ensure robustness and usability.

---

## 🚀 Features

- 📄 Create a new file  
- ❌ Delete an existing file  
- 📊 View file details (size, path, permissions, last modified)  
- ✍️ Write content to a file  
- 📖 Read content from a file  
- 📁 List all files in a directory  
- 🔍 Search for a file in a directory  
- 🪵 Error logging system (error_log.txt)  
- ⚠️ Exception handling for safe execution  

---

## 🧠 Concepts Used

- Object-Oriented Programming (Classes, Methods)  
- File Handling (java.io package)  
- Exception Handling (try-catch)  
- Collections (basic usage)  
- Modular Design

## ▶️ How to Run

### Using IDE (IntelliJ / Eclipse)
1. Create a new Java project  
2. Add all the `.java` files  
3. Run `Main.java`  

---

### Using Terminal

- Compile: javac *.java
- Run: java Main
